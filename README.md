## Kiwiland Trains 

### Build and Run

- Requires .net6.0

Run `dotnet build`

Run `dotnet run --project src\Kiwiland.Cli --routes AB5 BC4 CD8 DC8 DE6 AD5 CE2 EB3 AE7`

The routes above can be changed. The output based on the routes from the instructions
is always printed and the routes passed in the cli command will print after those.
